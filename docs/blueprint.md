# **App Name**: PalmistryAI

## Core Features:

- Photo Upload: User uploads a photo of their palm via a clear, intuitive interface.
- Palm Analysis: Gemini model analyzes the photo, identifying and categorizing major palm lines (heart line, head line, life line, etc.). The AI tool determines which elements in the photo are lines in the user's palm, as opposed to noise, and categorizes them using machine vision and palmistry techniques.
- Report Generation: AI generates a personalized report based on the analyzed palm lines. The report describes potential meanings of the lines based on palmistry.
- Line Highlighting: Visually overlay identified palm lines on the uploaded photo with distinct colors for each line type.
- Interpretation Key: Display color-coded interpretations corresponding to each highlighted line.

## Style Guidelines:

- Primary color: Soft, muted purple (#A080D0) to evoke a sense of mystery and intuition.
- Background color: Very light gray (#F0F0F0) to create a clean, modern look, similar to Apple's design aesthetic.
- Accent color: Dusty rose (#D0A0B0) for highlights and call-to-action buttons, providing a subtle contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use minimalist, line-based icons related to palmistry and intuition, in a style consistent with Apple's design language.
- Maintain a clean and spacious layout, emphasizing whitespace for a modern and intuitive user experience, akin to Apple's product pages.
- Subtle, smooth transitions and animations to enhance user engagement, reflecting Apple's polished design sensibilities.
